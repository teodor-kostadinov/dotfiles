# Teodor's Dotfiles

## Install

```bash
cd; curl -#L https://github.com/teodor-kostadinov/dotfiles/tarball/main | tar -xzv --strip-components 1 --exclude={README.md,LICENSE,bootstrap,.gitignore}; source .zshrc
```

## Shortcuts

- `OPTION` + `SPACE` = Alfred
- `OPTION` + `i` = Siri
- `OPTION` + `d` = Dash
